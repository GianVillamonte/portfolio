1 Purpose

This document presents a generalized and anonymized workflow for analysing protected flora monitoring data from repeated field surveys.

The workflow demonstrates how species-density observations can be processed, summarized and assessed through:

  • data preparation and filtering;
  • density comparisons by vegetation formation;
  • summary tables by species and formation;
  • temporal aggregation by survey period;
  • Mann-Kendall trend tests;
  • Sen’s slope estimates;
  • faceted visualizations of species-level density trends.

The dataset included in this repository is synthetic and anonymized. It preserves the structure of a typical flora monitoring dataset, but it does not reproduce project-level observations, site names, station names, client names, or confidential records.

The input table contains one row per survey record. The key fields are:

  • year: survey year;
  • season: survey season;
  • evaluation: numeric survey period, with half-year steps;
  • method: field survey method;
  • vegetation_formation: anonymized vegetation formation;
  • station_id: anonymized survey station;
  • species: anonymized protected species name;
  • density_ind_ha: density expressed as individuals per hectare.

3 Data preparation

The monitoring records are filtered to the target survey method, records with missing formation or density values are removed, and density is aggregated by survey period, species, vegetation formation and station.

flora <- flora_raw %>%
  filter(method == "Nearest-neighbor density survey") %>%
  drop_na(vegetation_formation, density_ind_ha) %>%
  group_by(year, season, evaluation, species, vegetation_formation, station_id) %>%
  summarise(density_ind_ha = sum(density_ind_ha), .groups = "drop") %>%
  mutate(
    season = factor(season, levels = c("Wet season", "Dry season")),
    vegetation_formation = factor(vegetation_formation),
    species = factor(species)
  )

flora %>%
  summarise(
    records = n(),
    years = n_distinct(year),
    survey_periods = n_distinct(evaluation),
    formations = n_distinct(vegetation_formation),
    stations = n_distinct(station_id),
    species = n_distinct(species)
  ) %>%

4 Density by vegetation formation

The first exploratory visualization compares species density across vegetation formations. A log-scale is used because density values usually vary strongly among protected flora species.

flora_positive <- flora %>%
  filter(density_ind_ha > 0)

6 Temporal trend dataset

Density values are averaged by species, vegetation formation and survey period before trend testing. A full survey-period grid is created so that missing combinations are kept as missing values rather than silently removed.

trend_data <- flora %>%
  group_by(year, season, evaluation, species, vegetation_formation) %>%
  summarise(mean_density_ind_ha = mean(density_ind_ha, na.rm = TRUE), .groups = "drop") %>%
  arrange(species, vegetation_formation, evaluation)

survey_grid <- expand_grid(
  evaluation = sort(unique(flora$evaluation)),
  species = levels(flora$species),
  vegetation_formation = levels(flora$vegetation_formation)

trend_data_full <- survey_grid %>%

7 Mann-Kendall trend analysis

The Mann-Kendall test is used to detect monotonic trends through time. Sen’s slope is used to estimate the direction and magnitude of change. Species-formation combinations with fewer than three observations are skipped.

mk_safe <- function(x) {
  x <- x[!is.na(x)]
  if (length(x) < 3) return(list(p_value = NA_real_))
  test <- trend::mk.test(x)
  list(p_value = unname(test$p.value))
}

sen_safe <- function(x) {
  x <- x[!is.na(x)]
  if (length(x) < 3) {
    return(list(sen_slope = NA_real_, conf_low = NA_real_, conf_high = NA_real_))
  }
  slope <- trend::sens.slope(x)
  list(
    sen_slope = unname(slope$estimates),
    conf_low = unname(slope$conf.int[1]),
    conf_high = unname(slope$conf.int[2])
  )
}
trend_results <- trend_data_full %>%
  arrange(species, vegetation_formation, evaluation) %>%
  group_by(species, vegetation_formation) %>%
  summarise(
    observations = sum(!is.na(mean_density_ind_ha)),
    p_value = mk_safe(mean_density_ind_ha)$p_value,
    sen_slope = sen_safe(mean_density_ind_ha)$sen_slope,
    sen_conf_low = sen_safe(mean_density_ind_ha)$conf_low,
    sen_conf_high = sen_safe(mean_density_ind_ha)$conf_high,
    trend_direction = case_when(
      is.na(sen_slope) ~ "Insufficient data",
      sen_slope > 0 ~ "Positive",
      sen_slope < 0 ~ "Negative",
      TRUE ~ "No change"
    ),
    significant = case_when(
      is.na(p_value) ~ "Not tested",
      p_value <= 0.05 ~ "Yes",
      TRUE ~ "No"
    ),
    .groups = "drop"
  ) %>%
  mutate(
    p_value = round(p_value, 3),
    sen_slope = round(sen_slope, 3),
    sen_conf_low = round(sen_conf_low, 3),
    sen_conf_high = round(sen_conf_high, 3)
  ) %>%
  arrange(species, vegetation_formation)

8 Density trends by vegetation formation

The plots below show monitoring observations across survey periods. Points represent station-level density observations. Lines show smoothed tendencies to support visual interpretation; formal trend direction is assessed through the Mann-Kendall and Sen’s slope table above.

  )

9 Interpretation notes

This workflow is designed for repeated biodiversity monitoring contexts where the objective is to evaluate whether protected flora species show meaningful changes over time or differences across vegetation formations.

A few methodological considerations are important:

  • zero values and sparse observations need careful treatment, especially when log scales are used;
  • trend tests should only be interpreted when enough repeated observations exist;
  • station-level variability can be high, so visual patterns should be interpreted together with formal trend statistics;
  • non-significant Mann-Kendall tests do not prove absence of change; they indicate that a monotonic trend was not detected given the available time series;
  • ecological interpretation should consider survey effort, method consistency, seasonality, disturbance history and species detectability.
